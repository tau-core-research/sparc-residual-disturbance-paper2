# Illustrative Rotation-Curve Points

This directory contains the minimal derived point table used only to render the classifier-context rotation-curve figure in the Paper 2 manuscript.

Files:

- `paper2_context_rotation_points.csv`

The table contains DDO154 as a correctly recovered regular class-A example and CamB as the named false-positive A outlier discussed in the failure-mode section. The rows combine SPARC rotation-curve columns with already-derived per-radius residual diagnostics from the Paper 1/Paper 2 packets.

The figure generated from these rows is visual context only. It shows how the Paper 2 residual classifier looks in rotation-curve space and must not be interpreted as an additional tuned endpoint or independent validation.
