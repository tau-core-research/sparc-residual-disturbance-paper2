# Illustrative Rotation-Curve Points

This directory contains the minimal derived point tables used only to render the illustrative rotation-curve figure in the Paper 2 manuscript.

Files:

- `ddo126_points.csv`
- `ddo50_points.csv`

These tables are copied from the Paper 3 seed packet and contain already-derived diagnostic columns for the DDO126 anchor and DDO50 control examples. They are not used by the Paper 2 primary endpoint, LOOGO classifier, shuffled-label null, bootstrap interval, baseline AUC comparison, or external-proxy gates.

The figure generated from these rows is visual context only. It shows how the residual-score families appear in rotation-curve space and must not be interpreted as an additional tuned endpoint or independent validation.
