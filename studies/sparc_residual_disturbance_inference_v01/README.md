# SPARC Residual-Disturbance Inference V01

This study directory is intentionally slimmed for the public Paper 2 repository.

It retains only:

- the final Paper 2 submission-source generator,
- the final derived packet tables required by that generator,
- the small derived Paper 1 inputs required for baseline comparison.

Exploratory scripts and branch packets used during development were removed from the public repository to keep the publication package focused.

Regenerate the submission source with:

```bash
python studies/sparc_residual_disturbance_inference_v01/make_paper2_submission_source_v01.py
```
